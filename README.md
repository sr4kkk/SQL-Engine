# 🚀 SQL Engine

**SQL Engine** é uma extensão poderosa e leve para o Visual Studio Code, projetada para desenvolvedores que desejam visualizar e gerenciar bancos de dados SQL sem sair do ambiente de desenvolvimento.

---

## ✨ Funcionalidades

* **Visualização Direta:** Explore tabelas, colunas e registros com uma interface limpa.
* **Execução de Queries:** Rode comandos SQL complexos e veja os resultados em tempo real.
* **Conexão Rápida:** Suporte para múltiplos bancos de dados (SQLite, MySQL, PostgreSQL).
* **Histórico de Consultas:** Acesso rápido aos comandos executados anteriormente.

## 🛠️ Tecnologias Utilizadas

* [TypeScript](https://www.typescriptlang.org/)
* [VS Code Extension API](https://code.visualstudio.com/api)
* [Node.js](https://nodejs.org/)

## 🚀 Como começar

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/sr4kkk/SQL-Engine.git](https://github.com/sr4kkk/SQL-Engine.git)
    ```
2.  **Instale as dependências:**
    ```bash
    npm install
    ```
3.  **Abra no VS Code:**
    ```bash
    code .
    ```
4.  **Inicie o Debug:**
    Pressione `F5` para abrir uma nova janela do VS Code com a extensão rodando.
